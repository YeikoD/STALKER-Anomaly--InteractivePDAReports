### 📢 Version 1.0.0 - Stable Release (16/07/2026)

> **IMPORTANT NOTE:** This is the first official stable release (v1.0.0) under our new versioning system. All previous test files are now deprecated. A **clean install** is highly recommended to avoid conflicts.

#### 🌟 NEW FEATURES

- **Dynamic PDA Locations:** The PDA now displays exact location descriptions relative to nearby landmarks (e.g., *"North of..."*).

- **Bilingual Cardinal Points:** Fully translated and integrated cardinal points in both **English and Spanish**.

- **Russian Language Support (Beta):** Initial localized localization added for Russian-speaking Stalkers.

- **Vengeance System:** Enemies of a fallen NPC will now actively seek revenge, making combat dynamics much more unpredictable.

- **Developer Debug Mode:** Integrated dedicated debug tools to make development, testing, and troubleshooting easier.

#### 🔧 BUG FIXES & REFACTORING

- **AI Behavioral Fix:** Fixed an issue where allies of a fallen enemy would defame them instead of defending their honor.

- **Radio Broadcast Fix:** Silenced non-sentient mutants and mindless entities (*mobs*) so they no longer transmit on public radio frequencies.

- **Russian Encoding Fix:** Re-encoded all Russian XML files to `Windows-1251` to prevent text corruption and broken characters in-game.

- **Standardized Versioning:** Implemented a new, clean semantic versioning system (v1.x.x) for future updates.
